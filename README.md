# noj2tab
NOJ to TAB dictionary format convertor which works in the following transition process.

    EPWING -> NOJ -> TAB -> OPF -> PRC(Mobi)

## Requirements
- Python 2.7
- Clime

## Usage
    $ python noj2tab.py main [--outfile=<default:'tabfile'>] [--delimit=<value>] <filename>
